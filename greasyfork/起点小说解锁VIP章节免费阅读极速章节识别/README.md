**本脚本仅限学习交流，请勿商业使用！**

该脚本用于解锁起点中文网的部分VIP付费小说，支持绝大部分小说的VIP章节自动解锁，非常舒爽

媲美原版付费阅读效果，完美兼容字体切换

小说找不到资源会自动切换书源，已经支持"58小说网","官术网","读书阁","笔趣阁"等书源站。在脚本菜单中或阅读页面设置中客手动指定书源
